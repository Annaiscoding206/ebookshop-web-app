# 📚 E-Bookshop Web Application

## 📌 Overview
This project is a Java-based web application for an online bookstore. It allows users to browse books, register/login, add items to cart, and place orders, with data stored in a MySQL database.

---

## ⚙️ Technologies Used
- Java (Servlets)
- HTML, CSS
- Apache Tomcat (web server)
- MySQL (database)
- JDBC (database connection)

---

## Key Features
- Browse and search books by author
- User registration and login system
- Password reset functionality
- Add to cart and view cart
- Remove items / clear cart
- Checkout with validation
- Display stock availability

---

## System Flow
1. User selects books from main page  
2. System retrieves book data from database  
3. User adds items to cart  
4. Order is processed and stored in database  

---

## Database
- Tables: `books`, `customers`, `order_records`
- Includes stock management and order tracking
